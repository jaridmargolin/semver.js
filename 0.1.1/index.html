Getting Started

Package Managers

npm: npm install semver.js

bower: bower install semver.js

semver

Lightweight module to compare semver versions.

compare

Compare two semver values. Optionally accepts param to force comparison based on passed specificity (major, minor, patch).

Function
compare(firstComparator, secondComparator, specificity)

Parameters

Name Type Default Description
firstComparator string The semver string to compare with.
secondComparator string The semver string to compare against.
specificity string patch major || minor || patch.

isGreater

Determine if semver a > semver b.

Function
isGreater(firstComparator, secondComparator, specificity)

Parameters

Name Type Default Description
firstComparator string The semver string to compare with.
secondComparator string The semver string to compare against.
specificity string patch major || minor || patch. 
semver.isGreater("1.0.0", "0.9.0");
// true

Contributing

In lieu of a formal styleguide, take care to maintain the existing coding style. Add unit tests for any new or changed functionality. Lint and test your code using Grunt.

Also, please don't edit files in the "dist" subdirectory as they are generated via Grunt. You'll find source code in the "lib" subdirectory!
